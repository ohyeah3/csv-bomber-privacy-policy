# Privacy Policy (プライバシーポリシー)

**CSV Bomber**（以下、「本拡張機能」といいます / hereinafter referred to as "this extension"）の提供にあたり、利用者のプライバシー情報の取り扱いについて、以下の通りプライバシーポリシーを定めます。

---

## 日本語 (Japanese)

### 1. 収集する情報および利用目的
本拡張機能は、利用者の個人情報（氏名、メールアドレス、閲覧履歴など）を一切収集・記録・送信しません。
拡張機能の動作に必要な設定データ（有効化されたドメイン一覧など）は、すべて利用者のブラウザ内（Google Chromeのローカルストレージ）のみに保存され、外部のサーバー等に送信されることはありません。

### 2. 権限（Permissions）の使用目的
本拡張機能は、その目的（ExcelでのCSV文字化け防止）を達成するために以下の権限を使用します。
- `storage`: ユーザーが機能を有効にしたサイトの設定をブラウザ内に保存するため。
- `activeTab`: 現在開いているタブのURLを確認し、機能の有効/無効の状態をポップアップに表示するため。
- `host_permissions` (`http://*/*`, `https://*/*`): ユーザーが任意に有効化したWebサイトにおいて、CSVのダウンロードを検知して自動的にBOMを付与するため。

### 3. プライバシーポリシーの変更
本拡張機能は、機能のアップデート等に伴い、本プライバシーポリシーをいつでも変更できるものとします。

- **作成日**: 2026年5月28日
- **開発者**: Ohyeah3

## English

### 1. Information Collection and Use
This extension does not collect, record, or transmit any personal information of users (such as name, email address, browsing history, etc.).
All configuration data required for the operation of the extension (such as the list of enabled domains) is stored exclusively within the user's browser (Google Chrome local storage) and is never transmitted to external servers.

### 2. Purpose of Permissions Used
This extension uses the following permissions to achieve its purpose (preventing CSV corruption in Excel).
- `storage`: To save the settings of sites where the user has enabled the extension within the browser.
- `activeTab`: To check the URL of the currently open tab and display the enabled/disabled status in the popup.
- `host_permissions` (`http://*/*`, `https://*/*`): To detect CSV downloads and automatically add a BOM on websites voluntarily enabled by the user.

### 3. Changes to this Privacy Policy
This privacy policy may be updated from time to time in accordance with extension updates.
