# Privacy Policy (プライバシーポリシー)

**CSV Bomber**（以下、「本拡張機能」といいます）の提供にあたり、利用者のプライバシー情報の取り扱いについて、以下の通りプライバシーポリシーを定めます。

## 1. 収集する情報および利用目的
本拡張機能は、利用者の個人情報（氏名、メールアドレス、閲覧履歴など）を一切収集・記録・送信しません。
拡張機能の動作に必要な設定データ（有効化されたドメイン一覧など）は、すべて利用者のブラウザ内（Google Chromeのローカルストレージ）のみに保存され、外部のサーバー等に送信されることはありません。

## 2. 権限（Permissions）の使用目的
本拡張機能は、その目的（ExcelでのCSV文字化け防止）を達成するために以下の権限を使用します。これらの権限はローカルでの処理にのみ使用され、不正な目的で使用されることはありません。
- `storage`: ユーザーが機能を有効にしたサイトの設定をブラウザ内に保存するため。
- `activeTab`: 現在開いているタブのURLを確認し、機能の有効/無効の状態をポップアップに表示するため。
- `host_permissions` (`http://*/*`, `https://*/*`): ユーザーが任意に有効化したWebサイトにおいて、CSVのダウンロードを検知して自動的にBOMを付与するため。

## 3. プライバシーポリシーの変更
本拡張機能は、機能のアップデート等に伴い、本プライバシーポリシーをいつでも変更できるものとします。変更後のプライバシーポリシーは、本ページに掲載された時点から効力を生じるものとします。
